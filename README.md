# CALIGO-GS-Low-Light-3DGS
这篇论文提出了 CALIGO-GS，一种针对低光照及曝光不均场景的 3D Gaussian Splatting (3DGS) 重建框架。 传统 3DGS 在低光条件下易受噪声和照明不足影响，导致几何结构崩塌和外观不一致。为解决此问题，本文引入了三个核心模块： 多视角频谱协同频率变换 (MSCFT)：在频域上解耦光照与结构信息，对齐跨视角的全局亮度并抑制高频噪声，恢复纹理细节。 几何敏感密度调节 (GSDR)：利用渲染深度图的梯度信息引导高斯原语的生长与修剪，在结构复杂处增加密度，在平坦区域减少冗余。 跨视角光度一致性校正网络 (CPC-Net)：通过几何重投影约束消除视角间的色差和亮度漂移。
